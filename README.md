# Идея компании
Человек фотографируется в туристической зоне исторического центра города. Система накладывает стилизацию под историческую газету/старую открытку этого города и сразу печататет фото. На обороте либо же внизу расположена реклама местных ресторанов/отелей/прочих услуг с QR-кодом или купоном. Заработок бизнеса обеспечивается франшизой и рекламой (ресторанов и прочих мест в городе).
